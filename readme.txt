== Description ==

WP Q&A Quiz is a easy to use plugin that extend your site with Questions & Answers quiz functionality.

Main Features:

* Refresh page after user answer question (you can set number of questions in admin)
* Set start image for every quiz
* Unlimited Quiz
* Unlimited Quiz questions/answers
* Facebook Share button
* Responsive design
* Result message settings (set result text & image based on user quiz result)
* Easy to use & manage

**Please use support forums if you have found any bug or have any other problem with plugin**

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-qa-quiz` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
